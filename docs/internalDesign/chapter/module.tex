\chapter{モジュール設計}
\section{前提}
我々は\texttt{Ruby on rails}を用いて開発を行う．この言語の慣習にのっとり，これ以降，
\begin{itemize}
    \item index
    \item new
    \item show
    \item edit
    \item create
    \item destroy
    \item update
    \item home
\end{itemize}
の8つは「アクション」と呼び，これ以外の関数をすべて「メソッド」と呼ぶ．\par
アクションはコントローラが異なれば同名のものを使用するため，モジュールIDを「コントローラ名．アクション名」
とする．また，1アクション1機能を持つ．
