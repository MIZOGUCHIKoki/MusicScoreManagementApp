		\item あると良い機能をあげてください．
		      \begin{itemize}
			      \item 各楽譜の利用楽器別に検索したい．
			      \item 「金管5重奏」などのジャンルを記述できるような汎用的なフィールドが欲しい．
			      \item 各楽譜の出版社がわかると探しやすい．
			      \item どこからでもアクセスしたい．
			      \item グレード別検索ができるようにしてほしい．
			      \item 作曲者・編曲者，演奏時間も知りたい．
			      \item 参考音源のURLを貼れる場所が欲しい．
